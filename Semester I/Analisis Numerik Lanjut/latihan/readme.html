Metode Bisection

Pemilihan selang ini menjadi hal terpenting bagi metode Bisection agar hasilnya konvergen.

Mari kita selesaikan:

# informasi awal
a = 0.25
b = 0.5
iter_max = 100
tol_max = 10^(-5)
i = 0

hasil = data.frame(n = NA,a = NA,b = NA,c = NA)

while(i <= iter_max && (b-a)/2 > tol_max){
  p = a + ((b-a)/2)
  FP = f(p)
  FA = f(a)
  FB = f(b)
  hasil[i+1,] = list(i,a,b,p)
  if(FA*FP < 0){b = p} else {a = p}
  i = i+1
}

Metode Newton Rhapson

Pada metode ini, mencari turunan pertama dari adalah sangat penting. Oleh karena itu, kita perlu melakukannya:

library(Ryacas)
eq = "(x-1)*Tan(x) + x * Sin(x*Pi)"
eq %>% y_fn("D(x)") %>% yac_str()
## [1] "(x-1)/Cos(x)^2+Tan(x)+x*Pi*Cos(x*Pi)+Sin(x*Pi)"

Kita telah mendapatkan . Setelah itu, kita akan set titik iterasi awalnya .

# initial condition
df = function(x){(x-1)/cos(x)^2+tan(x)+x*pi*cos(x*pi)+sin(x*pi)}
x_0 = 0.5
p = 100 # nilai dummy
i = 1

hasil = data.frame(iter = 0,
           p = x_0)

while(i <= iter_max && abs(p-x_0) > tol_max){
  p = x_0 - (f(x_0) / df(x_0))
  hasil[i+1,] = list(i,p)
  x_0 = p
  i = i + 1
}
