CURVE FITTING

Kita akan belajar dulu dari dasarnya. Kita juga akan lihat dan bandingkan cara pengerjaan dengan algoritma bikinan sendiri dan algoritma dar base R.

Linear Curve Fitting

Sering disebut sebagai regresi linear.

# ini data contoh
x = c(32,64,96,118,126,144,152.5,158)
y = c(99.5,104.8,108.5,100,86,64,35.3,15)

# we will make y the response variable and x the predictor
# the response variable is usually on the y-axis
plot(x,y,pch=19)

Untuk membuat model linear, kita bisa gunakan base R via lm(). Berikut adalah caranya:

# linear regression
model1 = lm(y~x)
model1

n = length(x)
sum_xi = sum(x)
sum_xi_xi = sum(x^2)
sum_xi_yi = sum(x*y)
sum_yi = sum(y)

A = matrix(c(n,sum_xi,sum_xi,sum_xi_xi),nrow = 2,byrow = T)
