Soal Latihan
================

# SOAL

Temukan akar persamaan berikut:

\[(x-1) \tan{x} + x \sin{\pi x} = 0, x \in [0,1]\]

Dengan keakuratan hingga \(10^{-5}\) menggunakan metode *Bisection* dan
*Newton-Rhapson*.
